Waarom noem je de eerste HTML-pagina; index.html?

De eerste HTML-pagina wordt vaak index.html genoemd omdat veel webservers automatisch naar dit bestand zoeken als je een website opent

Met welke HTML versie werken nu?

We werken nu met HTML 5

Met welke CSS versie werken we nu?

We werken nu met CSS 3

Hoe kan het dat een website er net iets anders uit ziet in een ander browser? (Egde, Chrome, Firefox, Opera)

Een website kan er anders uitzien in verschillende browsers door variateis in hoe elke browser CSS en HTML interperteert en implementeert

Welke browser is het populairst en heoveel procent RAM gebruikt deze browers?

De populairste browser is momenteel google chrome, die ongeveer 10-15% van het RAM-geheugen op een gemiddelde computer kan gebruiken.
